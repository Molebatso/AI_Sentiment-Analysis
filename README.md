# 🤖 AI Sentiment Analysis Tool

## 📌 Project Overview

The **AI Sentiment Analysis Tool** is a modern web application developed using **HTML, CSS, and JavaScript**.
The system analyzes user-provided text and classifies emotional sentiment into:

* ✅ Positive
* ❌ Negative
* ⚪ Neutral

The application also generates:

* Confidence scores
* AI-style explanations
* Emotional tone insights
* Interactive charts and analytics

This project demonstrates practical implementation of:

* Artificial Intelligence concepts
* Text classification
* Data visualization
* Frontend web development
* Dashboard design
* User interaction and analytics

---

# 🚀 Features

## ✅ Sentiment Classification

Analyze:

* Product reviews
* Tweets
* Customer feedback
* YouTube comments
* Movie reviews
* General text input

The system automatically determines whether the sentiment is:

* Positive
* Negative
* Neutral

---

## ✅ Confidence Score

Each analysis includes an AI-inspired confidence percentage showing how certain the system is about the detected sentiment.

Example:

* Positive – 92%
* Negative – 81%
* Neutral – 65%

---

## ✅ AI-Generated Explanation

The application generates a short explanation describing:

* Emotional tone
* Sentiment indicators
* Overall interpretation of the text

---

## ✅ Batch Analysis

Users can analyze multiple texts simultaneously by separating entries with new lines.

This is useful for:

* Large review datasets
* Social media monitoring
* Customer feedback analysis

---

## ✅ Sentiment History

The system stores all analyses during the current session using Local Storage.

Each history item contains:

* Original text
* Sentiment result
* Confidence score
* Timestamp

---

## ✅ Dashboard Analytics

The application includes:

* Pie chart visualization
* Bar chart analytics
* Statistics dashboard
* Total analysis tracking
* Sentiment count summaries
* Average confidence score

---

## ✅ Search and Filter

Users can:

* Search sentiment history
* Filter Positive entries
* Filter Negative entries
* Filter Neutral entries

---

## ✅ Export Features

The application supports:

* CSV Export
* PDF Report Export

This allows users to save analysis results professionally.

---

## ✅ Dark Mode

The project includes:

* Light Mode
* Dark Mode

for better accessibility and user experience.
